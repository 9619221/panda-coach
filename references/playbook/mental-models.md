## M1 · 道法术器（世界观骨架）

**定义**：万物四层结构 —— 从哲学到执行的完整链条。

| 层 | 内容 | 在交易里 |
|---|---|---|
| **道** | 哲学 / 认知 / 世界观 | 市场观、人性观、概率观、反 FOMO |
| **法** | 规则 / 原则 / 系统 | 60 分制、盈亏比 1:1.5、14 纪律 |
| **术** | 技术 / 战法 / 具体方法 | 12 金 K、价格行为、双龙、Vegas、谐波、缠论、SMC |
| **器** | 工具 / 指标 / 软件 | 布林带、MACD、RSI、EMA144-169、TradingView、OKX |

**核心论断**：多数散户死在"**只学器（指标软件），不修道（哲学认知）**"。

**熊猫用法**：遇到问"哪个指标最准"的学员 → 反问「你修道了吗？没修道学什么器」。

## M2 · K 线 > 形态 > 指标（优先级）

**定义**：最直接反应市场价格未来走势的**排序**。

```
K 线 > 形态 > 指标
  ↑      ↑      ↑
 最直接  次之  间接
```

**自用三指标**：布林带 / MACD / RSI
**品牌指纹**：EMA 144-169 **双隧道**（熊猫独门组合）

**熊猫反复强调**：「我很少用指标作为进场依据」 —— 指标是**参考**，不是**依据**。

## M3 · 武器论

**比喻**：指标是杀敌的武器，手枪、步枪、机关枪、大炮…… 你不可能上战场带所有武器，**选一个最擅长的**即可。

**延展**：
- 价格行为 = **正室**（地位最高）
- 谐波 / 缠论 / SMC = **妃子**（辅助，单独不能做主）
- 辅助武器独立进场时 → **仓位打 5 折**

**熊猫用法**：学员问"要不要学谐波缠论 SMC 全部学完" → 「妃子不能僭越正室，选一个精通」。

## M5 · 60 分进场制（核心执行框架）

**定义**：每项满分 20，**总分 < 60 直接拒绝开仓**。

| # | 维度 | 20 分满分条件 | 0 分拒绝条件 |
|---|------|-------------|-------------|
| ① | **趋势一致性** | 4/4 周期（日 / 4H / 1H / 15m）同向 | < 2/4 同向 |
| ② | **关键位** | 入场价 2× 止损距离内无强压力 / 支撑 | 距关键位 < 止损距离 |
| ③ | **信号质量** | 主信号命中 + 多周期共振 | 无主信号 → 封顶 5 分 |
| ④ | **交易计划完整性** | 方向 / 止损 / 止盈 / 仓位 / R:R 全齐 | 缺一减 5 |

→ 完整打分细则 / 信号四大类（12 金 K / 价格行为 / 指标 / 均线）见 [`trade-flow.md`](trade-flow.md) §Step 3

**熊猫类比**：「60 分像高考，偏科不行」。

## M5b · 盈亏比 × 胜率表（数学底层）

| 盈亏比 | 所需胜率 | 标注 |
|--------|---------|------|
| 1:1 | 60% | 难（高胜率难维持） |
| **1:1.5** | **40%** | **中（熊猫推荐）** |
| 1:2 | 33% | 中 |
| 1:3 | 25% | 易 |
| 1:5 | 20% | 易（比抛硬币还低！） |

**核心论断**：**宁要低胜率高盈亏比，不要高胜率低盈亏比**。
**数学逻辑**：长期期望值 = 胜率 × 盈利 - 败率 × 亏损。1:1.5 × 40% 胜率 = 正期望 0.5R / 单。

## M6 · 小周期入场，大周期方向

**定义**：
- **大周期**（日 / 4H）决定 **能不能开仓、开什么方向**
- **小周期**（1H / 15m）决定 **什么时候扣扳机**

**实操**：
1. 日线看牛 / 熊 / 震荡 → 决定偏多 / 偏空 / 观望
2. 4H 看趋势结构 → 确定顺势方向
3. 1H 找关键位 → 预设入场区间
4. 15m 等信号 K 线 → 精确时机

**熊猫用法**：学员问"我看 1 分钟图进场行不行" → 「1 分钟量能 = 0，周期太短，雷打不动禁止」。

## M7 · 反 FOMO 句式识别

**骗局 / 牛市顶部通用话术**：
> 「你已经错过了 XXX，后来你又错过了 YYY，现在你还想错过 ZZZ 嘛？！」

**反问测试**（Template A）：
> 「如果你不缺机会，即便 BTC 涨上天，你会焦虑吗？**不会的**，因为你有别的机会。」

**识别要点**：
- 话术三要素：**过去错过** → **现在又错** → **未来不能再错**
- 情绪挂钩：**恐惧**（错过 = 失败）+ **贪婪**（上车 = 翻身）
- 破解关键：**把"错过"重新定义为"避过一次可能的回撤"**

## M8 · 四象限时间管理（认知篇第 7 课）

| 象限 | 内容 | 策略 |
|------|------|------|
| I 紧急重要 | 盘中突发、急单 | 减少它（通过 II 区准备） |
| **II 重要不紧急** | **系统迭代 / 健康 / 学习 / 读书** | **最多时间投入** |
| III 紧急不重要 | 别人的需求 / 会议 / 消息 | **授权 / 拒绝** |
| IV 不紧急不重要 | 刷抖音 / 刷短视频 | **彻底避免** |

**熊猫用法**：学员问"我每天很忙但没进步" → 反问「你在哪个象限多？肯定 I + IV，II 区几乎没有」。

## M9 · 六锦囊妙计（认知篇第 7 课）

```
管碾压 → 高效睡眠 → 授权 → 专注 → 重复利用时间 → 放空放松
```

1. **管碾压**：管理精力碾压源（情绪 / 杂事 / 人际）
2. **高效睡眠**：睡眠是最大 ROI 投资
3. **授权**：能让别人做的不自己做（反"烈士陷阱"）
4. **专注**：单任务 > 多任务（反抖音化思维）
5. **重复利用时间**：通勤 / 等车 / 碎片时间 → 听书、复盘
6. **放空放松**：脑子也要休息，不然 II 区做不下去

## M10 · 殡仪馆思考法（认知篇第 7 课）

**定义**：想象你葬礼上，**亲人 / 朋友 / 同事 / 学员**怎么评价你？倒推今天该怎么活。

**实操三问**：
1. 你希望**亲人**说你是个怎样的家人？→ 倒推今天对家人的时间
2. 你希望**同事 / 学员**说你留下了什么？→ 倒推今天的工作价值
3. 你希望**你自己**回头看，没白活什么？→ 倒推当下的优先级

**熊猫定位**：这是破除"忙得有意义"幻觉的终极杠杆。
