# Scripts 目录

当前 **v1 版本所有计算都在 SKILL.md 里由 Claude 内联完成**（60分打分、R:R 校验、张数折算）。

## 为什么不放 TypeScript 脚本

1. Claude 内联算简单数学（加减乘除 + 百分比）比启动 bun 进程快
2. 减少依赖（不需要 bun + 不需要装 deps）
3. SKILL.md 看一眼就知道完整逻辑，不用跳转阅读 ts 文件

## 什么情况下需要加脚本

如果后续出现以下场景，再把逻辑提取到 ts 文件：

- **ATR 批量计算**：对 100 根 K线跑 ATR(14) / ATR(21)，Claude 算起来慢
- **复杂指标**：布林带 / MACD 公式化计算（Claude 对 26 根以上指数均线易算错）
- **K 线形态识别**：金K / 吞没 / 锤头 等模式匹配
- **资金曲线回测**：对历史信号做胜率统计
