# Panda Coach 回测引擎

按熊猫教练 V3.2 五步骨架实现的事件驱动回测器。数据来源 OKX 公开 REST API（无需密钥），策略对齐 [`trading-system/system_v3.md`](../../trading-system/system_v3.md)。

---

## 快速开始

```bash
# 1. 装环境（推荐 uv）
pip install uv  # 或 winget install astral-sh.uv
cd panda-coach/scripts/backtest
uv venv && uv pip install -e .

# 2. 拉数据 + 跑回测 + 出报告（默认 BTC-USDT-SWAP 近 365 天）
uv run python run.py --inst BTC-USDT-SWAP --days 365

# 跑过一次后用本地缓存
uv run python run.py --inst BTC-USDT-SWAP --days 365 --no-fetch
```

输出在 `reports/`：
- `<inst>_summary.md` —— 核心指标表
- `<inst>_trades.csv` —— 逐单明细
- `<inst>_equity.png` —— 资金曲线

## 撮合假设（贴近实盘但保守）

| 项 | 取值 | 说明 |
|---|---|---|
| 进场价 | 信号 K 的 close | 限价单近似（V3.2 §2.5 不挂整数关，本回测忽略） |
| 滑点 | 0 | 小账户 + 主流币忽略，结果略乐观 |
| 手续费 | 双边 0.05% | OKX SWAP 普通账户 taker 费率 |
| 单 K 内同时穿过止损止盈 | 按止损成交 | 保守，high/low 顺序未知 |
| 持仓上限 | 1 | V3.2 ≤3 简化 |
| 反向开仓 | 同日内不允许 | V3.2 §禁忌 5 |

## 退场红线（state machine）

| 条件 | 行为 | 解除 |
|---|---|---|
| 当日累亏 > 5% | 全停 | 24h |
| 同币止损 | 该币冷静 | 4h |
| 连续 3 单亏 | 复盘日 | 24h |
| 当周累亏 > 10% | 风险减半 | 1 周 |

---

## 已知偏差（结果偏乐观的方向）

1. **进场价用 close**：实盘限价单可能挂不到 close，价格滑过后未必成交
2. **关键位识别程序化 vs 人眼**：fractal 法找 swing 比人眼判断保守，可能错过部分关键位
3. **没有形态识别**：头肩 / 双重顶 / 反转 4 步全跳过 → 信号源比 V3.2 实际少
4. **量能阈值是拍的**：1.3× MA20 是经验值，不一定贴合各币种
5. **滑点 / 资金费率忽略**：永续合约持仓过夜会扣资金费

→ **结论**：回测正期望不等于实盘必赚，这是熊猫第 1 条原则「不预测，只验证数学」的当代版本。

## 首跑结果（BTC-USDT-SWAP，最近 365 天，2025-05 ~ 2026-05）

| 指标 | 值 |
|---|---|
| 总单数 | 232 |
| 胜率 | 41.8% |
| 实测 R:R | 1:1.84 |
| 盈亏因子 | 1.32 |
| 单笔期望值 | +0.246 U |
| 总盈亏 | +30.56 U |
| 收益率 | +97.0% |
| 最大回撤 | -38.4% |
| 期末余额 | 62.06 U（起始 31.5 U） |

**数学正期望成立** —— 胜率 41.8% × R:R 1.84 ≈ V3.2 §0.1 第 3 条「40% 胜率 × 1:2 = 长期正期望」。

### ⚠️ 必须打的警告（别拿这个数字盲目实盘）

1. **走势偏多头**：2025-05 ~ 2026-05 BTC 是「先震荡后单边上行」，对顺趋势策略天然友好；2022 熊市 / 2018 大跌应另测
2. **参数有调试痕迹**：1H ATR 阈值从 0.02 调到 0.005（V3.2 §1.5 原文是日线口径，迁到 1H 后必须降阈值），属合理调整但**未做敏感性测试**
3. **回撤 38%**：满足实盘退场红线（周累亏 >10% 减半仓位）会进入风险减半状态，这部分回测已实现；实测中触发了多次冷静期
4. **滑点 / 资金费率为 0**：永续合约持仓过夜会扣资金费，对长持单不利；实盘扣完收益会降一档
5. **进场用 close 价**：限价单实盘可能挂不到该价
6. **样本单一**：单币种、单年；推广前必须多币种 + 多年份验证

→ **结论**：管道跑通了、数学上印证了 V3.2 期望值理论；但「回测能赚 ≠ 实盘能赚」是熊猫第 1 条原则的当代版本。下一步要做敏感性测试 + 多币种 / 多年份回归。
