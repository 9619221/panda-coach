# 第一层｜道 · 世界观

## §1 交易宪法（每次开单前默念，V1.1 沿用）

1. 我不预测市场，只跟随结构
2. 我对手是主力 / 庄家，不是其他散户
3. 我靠数学盈利（40% 胜率 × 1:2.5 R:R = 长期正期望）
4. 没人是常胜将军 — 单笔亏损是成本
5. 反 FOMO：错过的行情比抓错的便宜

## §2 庄家自检 3 问（V1.1 沿用）

- 主力在这位置想吸还是出？
- 这个止损位会不会被插针扫掉？
- 突破是真突破还是诱多 / 诱空？

## §3 退场红线（V1.1 沿用 + V2.0 新增情绪识别）

任意一条触发即停手：
- 当日累亏 > 5% → 全停 24h
- 同币止损后 4h 冷静期
- 连续 3 单亏 → 复盘日（至少 24h 不开仓）
- 当周累亏 > 10% → 减半仓位 1 周
- 当月累亏 > 20% → 暂停至月末

## §4 V2.0 新增 · 心态识别 SOP

### 4.1 情绪化 7 信号（任 2 命中触发自检）
- 看盘超 4 小时 + 没单可下却放不下手机
- 错过一波后立刻想追下一波
- 止损后 1 小时内想加倍下单复仇
- 浮盈后想加仓（贪）
- 浮亏后想加仓摊低（恐）
- 觉得「这次一定不一样」
- 翻别人收益截图心跳加速

### 4.2 情绪化处理流程
1. 关盘 30 分钟（手机离开视线）
2. 写下当时想做的单子（不下，仅记录）
3. 30 分钟后回看，问自己 3 问（庄家自检）
4. 仍想做 → 仓位 ×0.5 + 杠杆 ÷2
5. 不想做 → 退场红线 +1（接近触发情绪退场）

## §5 V2.0 新增 · 庄家阶段判定流程（威科夫 ABCD）

每次开单前必跑（参 system-庄家思维）：

### 5.1 大背景判定（日/周线）
- A 阶段：恐慌抛售（放量大阴+长下影）→ 不做
- B 阶段：吸筹震荡（K 缩+量萎）→ 观察不做
- C 阶段：吸筹结束+震仓（Spring 窗口）→ 准备做多
- D 阶段：牛市回归（SOS+LPS）→ 持续做多

### 5.2 大背景反向（派发四阶段，镜像）
- A→B→C(UT 准备做空)→D(LPSY 最佳做空)

### 5.3 速查表（V1.1 系统继承）
- SOS / 中级震仓 / Spring 出现 → 吸筹做多
- SOW / 带量 UT 出现 → 派发做空

# 第二层｜法 · 决策骨架

## §6 市场状态前置（V1.1 沿用 + V2.0 数字化）

读日线 ATR(14) / 价格：
- < 1.5% → 强震荡市，启用 §11 4B 高抛低吸
- 1.5%-2% → 弱震荡 / 中性，仓位减半
- 2%-3% → 趋势启动，启用 §10 4A 60 分制
- > 3% → 强趋势 / 急速行情，警惕反转

## §7 三要素打分（V1.1 沿用，V2.0 数字化）

### 7.1 趋势 25 分（4 周期共振：日/4h/1h/15m）

每个周期判定标准（量化）：
- 多头：价 > EMA21 + RSI > 50 + MACD 柱 > 0
- 空头：价 < EMA21 + RSI < 50 + MACD 柱 < 0
- 中性：以上三条不全满足

得分：
- 4/4 同向 = 25
- 3/4 同向 = 18
- ≤ 2/4 = 0（直接 PASS）

### 7.2 关键位 25 分

价格距离最近 EMA200 / BB 中轨 / 前高低 / 中枢边缘：
- ≤ 0.5×ATR(1H) = 25
- 0.5-1×ATR(1H) = 15
- > 1×ATR(1H) = 0

V2.0 新增：必须显式列出
- 上方流动性池（BSL）：前高/24h 高/BB 上沿/EMA 上隧道/整数关
- 下方流动性池（SSL）：前低/24h 低/BB 下沿/EMA 下隧道/整数关
- 价值平衡（折价/溢价区）：4H 区间中点 — 做多只在折价区，做空只在溢价区

### 7.3 信号 25 分（V2.0 新增 8 套 system 触发矩阵）

主信号（必须命中至少 1 类，否则封顶 5）：
- system-12金K：吞没/启明黄昏/锤子/仙人指路/多方炮/三兵 = +5
- system-MACD：金叉死叉/顶底背离/8 形态(佛手/天鹅/小鸭/海底捞月) = +5
- system-布林带：开口喇叭/紧口反转/上破回踩中轨金 K = +5
- system-价格行为：FB/SB/RB 结构/简单回调金 K = +5
- system-庄家思维：Spring/SOS+LPS/JOC/UT+SOW = +5

辅助武器（仅在 60-70 弱信号区生效，加 ±5）：
- 缠论：三类买卖点 / 中枢突破 / 盘整背驰 3 后续
- 谐波：5 点形态 PRZ + 信号 K
- SMC：FVG / OB / BMS-SMS / 折价区
- 波浪：浪 3 启动 / 浪 4 黄金分割回撤

失败陷阱（命中扣分）：
- 压力位三兵/红三兵 / 一阳指压力位 = -5（80% 假突破）
- 弱背景+强信号第一次入场 = -5（价格行为 §2.4）

- 缺关键位 = -5（金 K 不在关键位 = pass）

### 7.4 计划 25 分

缺一减 5：
- 明确方向（多/空）
- 明确入场价（限价为主）
- 明确止损价（带关键位依据）
- 明确止盈三档（T1/T2/T3）

## §10 4A 趋势市战法（V1.1 沿用）

- 入场：限价挂关键位回踩，不追市价
- 信号触发距小时收盘 < 15 分钟（避免迟到）
- 仓位：共振 4/4 满仓 / 3/4 仓位 ×0.7 / ≤2/4 禁

## §11 4B 震荡市战法（V1.1 沿用）

- 接近区间上沿（高 -0.5%）→ 试空，目标中位
- 接近区间下沿（低 +0.5%）→ 试多，目标中位
- 仓位减半（保证金 1-2 U/单）
- 止损放区间外 0.5%

## §12 止损规则（V1.1 沿用 + V2.0 量化）

取以下二者较远者：
1. 关键位 ± 1.5×ATR(1H)
2. 入场价 ± 1.5%

V2.0 新增量化：
- ATR(1H) > 入场价 1% → 1.5×ATR 优先
- ATR(1H) < 入场价 1% → 入场±1.5% 优先
- 止损就是止损：不能改、不能扛、不能加仓救

## §13 止盈三档（V1.1 沿用 + V2.0 量化）

- T1 = +1R 距离，平 50% + 止损移到入场（保本）
- T2 = +2R 距离，平 30%
- T3 = +5R 距离 或 反向 SMS 出现 或 反向威科夫事件 → 移动止损

V2.0 新增量化：
- T3 上限 = min(+5R, 矩形量幅, 波浪浪 3 目标)
- 反向信号优先于固定档位
- 单线止盈兜底：取下一个关键位，若 R:R < 1:1.5 → 拒绝此单或重选入场

## §14 V2.0 新增 · 双单/分批入场结构

适合本金小+错过率高的场景：
- 60% 仓位：限价挂关键位回踩（主仓）
- 40% 仓位：突破确认追价（妃子仓）
- 主仓未触发即作废，妃子仓独立管理

## §15 V2.0 新增 · 撤单规则

- 限价单挂出后 4H 未成交 → 自动撤单（场景已变）
- 入场价跑出 0.5% 仍未触发 → 立即撤单
- 出现反向 SMS / 反向威科夫事件 → 立即撤单

## §29 资金加仓规则

仅允许金字塔正向加仓（不允许摊低补亏）：
- 浮盈 ≥ 1R + 趋势仍在 → 加仓 ≤ 原仓位 × 0.5
- 浮盈 ≥ 2R + 突破前高 → 再加 ≤ 原仓位 × 0.3
- 加仓后总仓位 ≤ 账户 30%（硬规）
- 严禁浮亏加仓（V1.1 禁忌 5）

## §30 月度回撤熔断

- 当月累亏 > 10% → 减半仓位 1 周
- 当月累亏 > 15% → 暂停 1 周 + 强制复盘
- 当月累亏 > 20% → 暂停至月末 + V2.0 系统审查
- 当月连续 3 周亏损 → 全停 + V2.5 系统重写
