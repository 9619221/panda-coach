## 1. 五步骨架（唯一流程，源自 09-03 §76-77）

```
道（世界观 + 庄家思维）
↓
Step 1: 分辨趋势（参考 1H）       → 多/空/震荡 + 初/中/末期
Step 2: 找关键位（交易 15m）      → 计算 R:R 不达 1:1.5 放弃
Step 3: 找入场信号 + 假突破过滤   → 4 信号族任一族用熟
Step 4: 制定交易计划（十要素）    → 七行清单输出
Step 5: 执行                       → 浮盈/浮亏/停滞 三场景应对
Step 6: 复盘                       → 技术/仓位/心态 三栏
```

## Step 0 · 道（世界观）

### 0.1 五条交易宪法（每次开单前默念）
1. 不预测市场，只跟随结构
2. 对手是主力 / 庄家，不是其他散户
3. 靠数学盈利（40% 胜率 × 1:2 R:R = 长期正期望）
4. 没人是常胜将军 — 单笔亏损是成本
5. 反 FOMO：错过的行情比抓错的便宜

### 0.2 退场红线（任一触发即停手）
- 当日累亏 > 5% → 全停 24h
- 同币止损后 4h 冷静期
- 连续 3 单亏 → 复盘日（≥24h 不开仓）
- 当周累亏 > 10% → 减半仓位 1 周

### 0.3 庄家自检 3 问（开单前过一遍）
- 主力在这位置想吸还是出？
- 这个止损位会不会被插针扫掉？
- 突破是真突破还是诱多 / 诱空？

## Step 1 · 分辨趋势（参考周期 1H）

### 1.1 输出：多 / 空 / 震荡 + 初 / 中 / 末期
- 震荡（条形码）→ 直接 STOP，不进入后续（09-03 §88-89）
- 末期顺势 → 不做，反向观察
- 初 / 中期顺势 → 进入 Step 2

### 1.2 四大判断方法（选一研究透，09-03 §111-115）

| 方法 | 判断要点 |
|---|---|
| 裸K | 接近关键位距离 / 回调次数 / 回调深度 / 斜率 / 距关键位动能 |
| 均线 | 形态（破线/拐头/交叉/排列）+ Vegas 隧道 + 真假突破 |
| 趋势线 | 一二三法则（破线 → 不破前高/前低 → 突破前高/前低） |
| 形态 | 反转形态（头肩/微型/双重/三重/圆弧）— 中继形态不用 |

禁忌：2B 不能作分辨趋势依据（09-03 §114-115）

### 1.3 反转 4 步法（11 篇逐K复盘）
1 破趋势线 → 2 回踩均线 EMA20 → 3 形成窟窿（缩量回踩） → 4 突破前极值点
4 步缺一不可，否则只是「回调」非反转

### 1.4 参考周期出场触发（硬规）
参考周期趋势被破坏（趋势线破 / 均线交叉 / 颈线跌破）→ 清仓或减仓（09-03 §82-83）

## Step 2 · 找关键位（交易周期 15m）

### 2.1 三类共识点
- 水平关键位：前高 / 前低 / 极值点 / 反复横盘区
- 均线：EMA20 / EMA144-169 双隧道 / BOLL 中轨
- 通道线：趋势线 / 楔形 / FVG 缺口

### 2.2 使用原则（09-03 §121-126）
- 主要支撑位：只做多不做空 / 只平空不平多
- 主要压力位：只做空不做多 / 只平多不平空

### 2.3 反转形态颈线目标倍数（09-03 §127-129）

| 形态 | 颈线到极值倍数 |
|---|---|
| 头肩顶/底 | 1–2 倍 |
| 微型顶/底 | 1 倍 |
| 双重顶/底 | 1–3 倍 |
| 三重顶/底 | 2–3 倍 |

### 2.4 R:R 闸门（02-03 §22 / 02-13 §67-68）
- 入场 → 第一阻力（多）/ 第一支撑（空）= 止盈空间
- 入场 → 关键位外侧 0.2% = 止损空间
- R:R < 1:1.5 → 直接放弃此单（熊猫底线）
- 标准目标 1:2，大趋势放到 1:3+ 留底仓搏 1:5（02-13 §88）

### 2.5 区间不是一条线（09-01 §209-211）
建仓成本是一个区间，整数关口会被庄家差一丢丢
想做 76,000 就挂 75,950 或 76,050，不挂在整数关口

## Step 3 · 找入场信号 + 假突破过滤

### 3.1 四大信号族（09-03 §137-148）

| 信号族 | 做多 | 做空 | 单独使用 |
|---|---|---|---|
| 金K（12 金K） | 启明星 / 看涨吞没 / 垂形 / 刺透 | 黄昏星 / 看跌吞没 / 流星 / 乌云盖顶 | ✓ |
| 均线 | 金叉 / 多头排列 / 真破上隧道 | 死叉 / 空头排列 / 真破下隧道 | ✓ |
| 形态 | 突破上颈线 | 跌破下颈线 | ✓ |
| 量能 | 关键位放大量 | 关键位放大量 | ✗（必须配其他） |

双查要点（09-03 §138-140）：
- 变形：金K 形态变形则降级
- 位置：金K 不在 Step 2 关键位附近 → 作废按普通K 处理

### 3.2 必 2+ 加 2 法则（02-03 §93-138）
- 必 2 条件：关键位 + 高 / 低位极值
- 加 2 项以上（任选）：形态明显 / 假突破 / 吞没幅度 / 顺大趋势 / 盈亏比
- 必 2 + 加 2 ≥ 1 → 可开仓
- 多族共振得分更高，胜率更高

### 3.3 假突破过滤（最大陷阱）

| 判据 | 说明 |
|---|---|
| ① 3 根 K 线不向反方向发展 | 长度 = 交易周期，15m × 3 = 45min |
| ② 伴随放大量 | 明显巨量 |

~~③ 突破幅度 > 3%~~ → 熊猫当场删除，不执行

### 3.4 辅助武器（妃子规则）
仅在主信号边缘场景作 tie-breaker：
- 缠论：1 / 2 / 3 类买卖点 + 中枢突破 + 背驰
- 谐波：AB=CD / 加特利 / 蝙蝠 / 蝴蝶 / 螃蟹（两发子弹规则）
- SMC：流动性扫荡 / OB / FVG / PO3
- 波浪：3 浪主升

辅助单独入场 → 仓位打折（妃子不能僭越正室，具体打折看场景，原则上减半起步）

## Step 4 · 制定交易计划（十要素，09-03 §173-200）

| # | 要素 | 规则 |
|---|---|---|
| 1 | 标的 | ≤3 个合约（已固化） |
| 2 | 理由 | ≥2 条，可叠加（关键位 + 形态 + 金K + 量能 + 反复验证） |
| 3 | 仓位 | 先想"准备亏多少钱"（≤1.5U），再想赚多少 |
| 4 | 方向 | 单方向集训锁定，多空二选一 |
| 5 | 杠杆 | 能量守恒：杠杆高仓位轻；单笔保证金 ≤ 账户 20% |
| 6 | 入场点 | 建仓点 + 平均成本（分批必算） |
| 7 | 止损 | 统一 / 分批 / 移动 三选一；距关键位 ≥ 0.2%；进场前挂好不可改 |
| 8 | 止盈 | 统一 / 分批；R:R ≥ 1:1.5；保二争五（02-03 §22-24） |
| 9 | 对策 | 急涨急跌打止损 → 反手 / 反弹单 |
| 10 | 后手 | 结束后：复盘 / 休息 / 复仇 / 反向 |

### 4.1 七行清单输出格式
```
币种:    XXX-USDT-SWAP
方向:    做多 / 做空
数量:    N 张（合约乘数 X）
杠杆:    Yx（保证金 Z USDT）
入场:    价（限价挂单 / 区间内任意位置，不挂整数关）
止损:    价（距入场 -A%，亏损 -B USDT，依据=关键位 X）
止盈:    价（距入场 +C%，盈利 +D USDT，R:R = 1:E.E）
```

### 4.2 分批进场三方案（09-01 §207-242，可选不强制）
- 3+2+5：第 1 单 30% / 第 2 单 20% / 第 3 单 50%
- 3+3+4：第 1 单 30% / 第 2 单 30% / 第 3 单 40%
- 3+(动态补)：底仓 30%，后续按市场反应补
- 熊猫本人原话：「我很少加仓；金字塔加仓法很少用」（09-03 §280）
- 小账户（< 100U）OKX 最小张数限制下，分批难做，可全仓一次进，挂在区间内任意位置即可

### 4.3 三三三原则 + 投机一资金分配（术-仓位管理 §1.2 / 09-01）
- 总资产 1/3 现金（保命金）
- 1/3 长期资产（不动）
- 1/3 投机金（可亏，分配到交易）
- 投机金中的 60% 现金月到手口径上限
- 单笔风险 ≤ 投机金 1-2%

## Step 5 · 执行（三场景应对矩阵，09-03 §208-229）

| 场景 | 推荐动作 |
|---|---|
| 正向（浮盈） | 移动止损 + 分批止盈为主；金字塔加仓需技术+心态+运气三全 |
| 反向（浮亏） | 等打止损为主；逆势加仓先进先出、慎用；FIB 61% / EMA 死叉 / Vegas 真假破做参考 |
| 停滞（横盘） | 浮盈减仓落袋；浮亏减仓少亏；熊猫"几乎不拿着" |

### 5.1 14 条交易纪律（09-02 完整版）

执行层 7 条：
1 不追无背景支撑的突破
2 大背景空头只做回调多，不追新高
3 每根 K 形成后再看下一根（收线大于天）
4 形态不完整不进场（等 4 步反转完全）
5 逆势单失败，第 1 次信号清仓
6 顺势单在中线位置或新形态时减仓
7 浮亏不加仓（交易区间除外）

心态层 7 条：
8 浮盈才敢加（趋势确认后）
9 止损不动摇，极值点是底线
10 进场前已定止损，不能临场改
11 进场后大脑只能有三个声音（到止损 / 到 T1 / 到 T2）
12 不要看资金涨跌动手，看图说话
13 不要因为亏了就想赚回来（数学是月度维度）
14 永不逆势 — 100% 信心可有，100% 行情没有（09-02 §105-107）

### 5.2 进场后大脑三个声音（02-03 §22 / 术-止损止盈 §5.2）
1 价格到止损位 → 自动平仓
2 价格到 T1 → 平 75% + 底仓移止损到开仓价（保二）
3 价格到 T2 / T3 → 全平 / 移动止损搏更大（争五）
任何其他声音都说明你刚才分析得不够稳

### 5.3 自律保障（执行不下去时）
1 纪律自律：按 14 条机械执行
2 转移焦点：吃鸡 / 撸啊撸 / 蹦极
3 课金法：违纪赔同学钱（09-03 §230-234）

## 附录 A：熊猫本人配置示例（09-03 §261-283，仅参考）

熊猫的合约系统：
- 标的：3 个以内（同时盯 3 个不超）
- 工具：只用裸K（不用指标）
- 战法：「平霸战法」（4H 关键位 + 12 金K 入场）
- 止损：移动止损为主
- 仓位：分两批或全仓
- 加仓：很少加仓，金字塔法很少用

熊猫的现货系统：
- 标的：5 个以内
- 周期：日 / 4H 趋势 + 4H 入场
- 方向：只做多不做空
- 持仓：底仓过夜常态
- 止损：宽止损 + 不挂止损委托单（反弹再出）
- 核心目标：不由盈转亏（优先保浮盈）

## 附录 B：现货特殊规则（09-03 §284-302，用户当前不用）

- 不挂止损委托单 — 反弹再出
- 不由盈转亏 — 浮盈减仓优先
- 标的远离选品 — 不做自己看不懂的币
